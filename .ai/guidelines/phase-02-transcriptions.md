# Phase 02 Transcriptions Guideline

- `Transcription` is a child of `ContentItem`.
- Public listings remain item-based.
- Effective/main transcription resolution is: featured published transcription, then latest published transcription, then none.
- Items without an effective/main published transcription are not public.
- Draft transcriptions are hidden publicly.
- Markdown remains canonical; parsed timestamps/speakers are derived data.
- Public Markdown rendering must use the centralized safe renderer.
