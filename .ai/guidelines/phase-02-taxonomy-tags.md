# Phase 02 Taxonomy and Tags Guideline

- Categories are custom hierarchical models.
- Tags use Spatie tags later and remain flat.
- Do not install packages during documentation/planning tasks.
- Public tags must be scoped to the `content` type and enabled before display.
- Do not use unscoped free-form tag inputs in admin forms.
- Group categories are inherited by items for public filtering/display.
- Parent category filters include descendants.
- Category and tag landing pages list public `ContentItem` records only.
