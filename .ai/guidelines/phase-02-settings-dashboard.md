# Phase 02 Settings and Dashboard Guideline

- Use Spatie Settings later for typed global settings.
- Use normal database records for ordered visible homepage sections.
- Item pinning is separate from homepage section management.
- Dashboard widgets are editorial counts and warnings only.
- Do not add analytics, search logging, observability infrastructure, or retry dashboards.
- Dashboard links should point to Filament resource URLs rather than hard-coded route names.
