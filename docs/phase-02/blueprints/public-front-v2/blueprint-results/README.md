# Blueprint Results

These files are Filament Blueprint-style implementation plans generated from the Public Front v2 source blueprints.

Laravel Boost was used for installed-version context and version-aware documentation. The installed Filament Blueprint package does not expose an Artisan generation command in this checkout, so the vendor planning docs under `vendor/filament/blueprint/resources/markdown/planning/` were used to shape these manual plan files.

- `02-card-template-builder-plan.md`

- `12-implementation-sequence-plan.md`

## Use

Read `12-implementation-sequence-plan.md` before using any individual result plan. Do not start implementation until the active prompt explicitly asks for it and resolves the open decisions listed in the sequence plan.
