# Documentation Index

Active project context is intentionally small.

- `docs/phase-02/` contains the active Phase 02 specs, blueprints, tooling notes, and coverage matrix.
- `docs/research/` contains active research used by current planning.
- `docs/planning/` contains planning templates and recommended evergreen context files.
- `docs/archive/` contains historical documents and prompts. Archived files are not active instructions unless a current prompt explicitly references them.

The root Bootstrap Slice 0 documents were archived because they describe completed historical scope and contain exclusions that no longer match the current roadmap.

Active prompts should reference durable `.ai/guidelines/*.md` names, Phase 02 specs, and Phase 02 blueprints directly. One-time cleanup prompts belong in `prompts/archive/phase-02-superseded/` after use.
